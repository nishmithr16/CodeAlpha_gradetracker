Student Grade Tracker

This Java applet allows you to track student grades and generate reports.

Your browser does not support Java applets or Java is disabled.

Instructions:

  • Enter student name and marks (0-100)
  • Click "Add Student" to store the data
  • Click "Show Report" to view all students and statistics
  • Click "Clear All" to reset all data
  • Use Enter key to navigate between fields

Features:

  • Input validation for marks (must be 0-100)
  • Grade calculation (A, B, C, D, F)
  • Statistics: average, highest, lowest scores
  • Grade distribution report
  • Clean AWT-based user interface
